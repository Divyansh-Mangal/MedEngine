# MedEngine

The connective tissue a healthcare-innovation ecosystem is missing — and why conscious experience is the only thing it is ultimately for.

This repository holds two cross-linked documents. **`MedEngine.html` is the main paper.**

| File | What it is |
|---|---|
| **`MedEngine.html`** | The White Paper — the full argument, from the moral axiom down to the contract clauses. Start here. |
| `i4KIDS_India_DPR.html` | A live field deployment — the MedEngine spine re-scoped for paediatrics as *i4KIDS India*. |

The two documents link to each other (the White Paper points to i4KIDS as its field deployment; the DPR inherits the spine).
